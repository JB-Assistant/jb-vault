# Skill Curator

## Summary

A background maintenance system for agent-created skills. It tracks usage, moves unused skills through active/stale/archived states, and runs an auxiliary-model review to propose patches or consolidation.

## Why it matters

Skill creation is valuable only if the catalog stays useful. Curator lets agents save reusable procedures aggressively while still controlling token bloat, drift, and duplicate skills.
