## Core Thesis

The biggest unlock in Claude productivity isn't advanced prompting — it's using the built-in tools that 99% of people pay for but ignore.

## Four Tiers of Tools

### Tier One: Core Interface Tools

**Projects**
- Most important feature most people never set up properly
- Without Projects, every conversation starts from scratch
- Take 20-30 min to build context files and upload to dedicated Projects

**Claude Skills**
- Pre-loaded instruction sets saved as markdown files
- Call them in any chat/Project and Claude follows the instructions instantly
- Build once, reuse forever (e.g. Brand Voice Skill)
- Create: Customize → Skills → Enable Skill-Creator → tell Claude what you want

**Memory**
- Claude builds memories about you passively across conversations
- Go to Settings → Memory and review/delete/add what's retained
- Pro Tip: export memory from ChatGPT by asking it for a transfer document

**Connectors**
- Let Claude access your existing tools directly in any conversation
- Settings → Connectors: Google Drive, Gmail, Slack, Notion, Google Calendar, Excalidraw + 50+
- Author uses all six listed above

---

### Tier Two: Research & Thinking Tools

**Deep Research**
- Breaks down queries, searches dozens of sources, cross-references, returns cited report
- Takes 5-45 minutes depending on complexity
- Enable: click "+" in main chatbox

**Extended Thinking**
- Activates deeper reasoning for complex problems (strategy, analysis, multi-step)
- Trigger via language ("think deeply before responding") or toggle in model selection
- Pair with Opus 4 for complex deep reasoning

**Artifacts**
- Code files (HTML, React, documents, diagrams, spreadsheets) as separate editable files
- Pair with custom Skills for maximum power
- Ask: "create this as an Artifact" or "build as a downloadable file"

---

### Tier Three: Agentic Tools

**Claude Cowork** (Desktop app only)
Three daily-use features:
1. **Scheduled Tasks** — run tasks automatically on fixed cadence (daily research, scan Gmail/Calendar for briefs)
2. **File Access** — gives Claude access to desktop files/folders — Claude can edit and work in real files
3. **Plug-Ins** — package multiple Skills into a single role (more powerful than Skills alone)

**Cowork Dispatch**
- Run Claude Cowork on your phone
- Leave laptop running, prompt from anywhere — "like a mini OpenClaw"
- Found under "Dispatch" in Cowork settings

**Claude in Chrome**
- Desktop app users can start a task in Claude and have it work in browser without switching windows
- Chrome Web Store link in article

---

### Tier Four: Coding & Building

**Claude Code**
Capabilities:
- Complex debugging & code errors
- Ship websites/apps
- Write tests and handle testing workflow
- Run security audits
- Plan coding sessions
- Code in natural language (vibe coding)

**Slash Commands**
- Built-in command system for faster operation
- Can build custom slash commands

**CLAUDE.md**
- Markdown file at project root that Claude Code reads before every session
- Contains project-specific instructions (coding standards, architecture decisions)
- "Single biggest quality improvement most Claude Code users never set up"
- Create: make `CLAUDE.md` in project root

**Multi-Agent Mode / Subagents**
- Spin up multiple subagents working in parallel on different parts of same project
- Example: one writes tests while another builds the feature
- Claude Code auto-decides when to use subagents based on task complexity
- Give high-level outcome-oriented prompts rather than step-by-step instructions

**Memory (/memory)**
- Claude Code's own memory system (separate from web Claude memory)
- Tell it to remember project specifics, preferences, coding standards
- Combined with CLAUDE.md → compounding understanding over time

---

## Key Takeaways

1. Set up Projects with context files before anything else
2. Build Claude Skills for every recurring workflow
3. Enable Connectors for tools you actually use
4. Use Deep Research for comprehensive research tasks
5. Enable Extended Thinking + Opus 4 for complex reasoning
6. Use Cowork for scheduled/automated workflows
7. Set up CLAUDE.md in every project root
8. Use Claude Code's /memory for project-specific context

## Author's Bottom Line

"If you implement these four tips — slash commands, proper CLAUDE.md management, sub-agent triggers, and proper memory management — your Claude Code productivity will reach a new level."
