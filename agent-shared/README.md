# Agent Shared Folder

Shared workspace for Hermes and OpenClaw (CortexClaw) collaboration.

## Structure
- `plans/` — Active project plans and implementations
- `projects/` — Shared project notes and specs
- `memory/` — Cross-agent memory dumps (Layer 2 sync)
- `inbox/` — Items waiting for the other agent

## Rules
- Write here first, push to GitHub when done
- Tag entries with `[Hermes]` or `[OpenClaw]` so it's clear who posted
- Check inbox before starting big tasks
- Use dated filenames: `YYYY-MM-DD-title.md`

## Sync
Both agents read/write here directly. Sync happens via GitHub push/pull — make sure to commit when done.
