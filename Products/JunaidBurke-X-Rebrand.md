## Content Strategy — 3 Tweets Daily

### The Mix (for maximum growth)

Every day, post 3 tweets across these 3 categories:

| Tweet | Type | Goal | Length |
|---|---|---|---|
| **Tweet 1** | Value / Tip | Save people time, get saves | 5-8 bullet points, punchy |
| **Tweet 2** | Personal POV / Story | Build know-like-trust | 1-2 paragraphs, authentic |

## Tweet Template Library

### Tweet 1: Value Tip (Post 7-8am or 12pm)

**Format:** [Number] things [audience] should know about [topic]

**Examples:**

```

1. Context files aren't optional — they're the whole thing
2. The skill description is the trigger — write it like a trigger
3. More instructions ≠ better output
4. Test for 1000s of invocations, not just your 3 test prompts
5. Negative cases matter as much as positive ones
6. Scripts > step-by-step instructions
7. If it only works for you, it's overfit

More where this came from → follow.
```

```
How I structure a Claude agent in 5 minutes:

1. Define what it IS and what it's NOT
2. Write the description like a search query
3. Keep body under 500 lines
4. Scripts in /scripts, refs in /references
5. Test with bad inputs first

That's the whole thing.
```

## Engagement Strategy (Must-Do Daily)

Posting alone won't grow your account. You need to engage first.

**Daily minimum:**
- Reply to 5-10 posts from people in your niche
- Comment with a substantive take, not just "great thread"
- Quote-repost with an add, not just a like
- Respond to every reply and DM within 2 hours

**Who to engage with:**
- Search "Claude Cowork setup" / "AI agent skills" / "AI productivity" and reply to posts from the last 1-3 days
- Target accounts with 500-10K followers (big enough to matter, small enough to reply)
- Ignore blue-checkmarks with 500K+ followers — they won't reply
