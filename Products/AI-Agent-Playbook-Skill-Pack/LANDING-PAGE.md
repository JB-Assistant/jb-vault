## FAQ

**Do I need to be technical?**
No. The context files are fill-in-the-blanks. If you can write about yourself, you can fill these in. The agent can draft 80% of it for you automatically if you've got existing docs.

**What if I'm not one of the four personas?**
The structure generalizes. Take any pack, adjust the labels, it works for any professional.

**Does this work with anything other than Claude?**
The context file + SKILL.md format is the open standard used by Claude Cowork, Claude Code, and increasingly other agents. It's the most widely compatible format right now.

**How long does setup take?**
20 minutes for a basic setup. 45 minutes if you customize everything.

**What's your refund policy?**
If you set it up and it doesn't work for you, reach out. We'll figure it out.
