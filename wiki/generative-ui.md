# Generative UI

## Definition
Generative UI is the pattern where an AI agent composes, selects, or generates interface elements at runtime based on user intent, task state, and available actions, rather than only returning text or relying on static prebuilt pages.

## Key Ideas
- Software is splitting into two surfaces: headless/API/CLI interfaces for agents, and generated/composed interfaces for humans.
- Humans still need visual surfaces for trust, approval, editing, understanding, and control.
- Generative UI can range from controlled existing components to fully open, on-demand generated interfaces.
- Text-only chat is often insufficient for production agent products.
