# AI Agent Playbook — Skill Pack

## What's in this vault

This vault contains pre-built **Context Files** and **Skills** for your AI agent (Claude Cowork, Claude Code, or any SKILL.md-compatible agent).

---

## Part 1: Context Files (Persona Packs)

Each persona pack contains 3 files:

| File | Purpose |
|---|---|
| `about-me.md` | Who you are, what you do, who you serve |
| `voice.md` | How you communicate — style, words to avoid, samples |
| `preferences.md` | How the agent should work with you |

**4 persona packs included:**

1. **`founder/`** — Startup founders, CEOs, solo entrepreneurs. Fundraising, investor updates, team comms, strategy.
2. **`marketer/`** — Marketers, growth hackers, content creators. Campaigns, social posts, briefs, analytics.
3. **`consultant/`** — Freelancers, advisors, coaches. Client deliverables, proposals, meeting prep.
4. **`developer/`** — Engineers, architects, technical leads. Code review, PR descriptions, tech docs.

**How to use:**
1. Pick your persona folder
2. Fill in the `[BRACKETS]` — the agent does 80% of this for you if you connect Google Drive first
3. Save all 3 files in one folder
4. Select that folder at the start of every session
5. Optional: paste the condensed version into Global Instructions

---

## Part 2: Skills

4 ready-to-use `SKILL.md` templates:

| Skill | What it does |
|---|---|
| `meeting-prep.md` | Before every meeting: pulls context, generates briefing |
| `content-research.md` | Monitors sources, summarizes weekly |
| `client-brief.md` | Turns a topic into a structured client-ready brief |
| `slide-builder.md` | Runs the full Research → Brief → Generate pipeline |

**How to use:**
1. Copy the `.md` file to your skills folder
2. Customize the `[BRACKETS]` sections
3. The agent will activate it automatically when the trigger conditions match

---

## Recommended First Setup

1. Pick your persona pack
2. Connect your tools (Google Drive, Slack, Notion — see guide)
3. Drop source material into a folder, ask the agent to scan it and draft your context files
4. Edit what it gets wrong
5. Save to your context folder, select it in your agent

Time: ~20 minutes. Then your agent knows who you are, how you communicate, and what you expect.
