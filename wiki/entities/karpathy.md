# Andrej Karpathy

## Overview
AI researcher, former Tesla Autopilot director, and prolific educator. Creator of micrograd, distill.pub, and the [[karpathy-llm-wiki|LLM Wiki pattern]] that inspired a generation of agent memory systems. Active on X/Twitter @karpathy.

## Key Contribution
Published the viral "LLM Knowledge Bases" GitHub Gist that introduced the concept of using LLMs to build compounding personal knowledge bases as plain Markdown wikis — the foundation for the [[4-layer-memory-architecture]] used by OpenClaw, Hermes, and many others.

## Known For
- micrograd — tiny autograd engine
- Neural Networks: Zero to Hero (YouTube series)
- Tesla Autopilot ML leadership
- Building and sharing LLM knowledge bases at scale (~100 articles, 400K words)

## Relationship to This System
The [[karpathy-llm-wiki]] concept he authored is the theoretical foundation for the entire 4-layer memory architecture. Multiple community implementations (Alex Finn, AI Edge, OpenClaw/Hermes) trace their approach back to his pattern.
