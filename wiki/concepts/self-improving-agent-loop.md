# Self-Improving Agent Loop

A self-improving agent loop is the pattern where completed tasks, corrections, and failures are converted into reusable procedures/skills so future sessions improve without the human manually authoring every lesson.

## Pattern

1. Agent attempts task.
2. Agent hits friction or receives correction.
3. Agent completes the work.
4. Agent extracts the reusable method.
5. Agent saves/updates a skill or playbook.
6. Future runs load that procedure automatically.
