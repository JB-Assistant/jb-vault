# OpenRouter

## Overview
OpenRouter is a unified API gateway for LLMs, providing access to 500+ models across major providers with single billing, automatic failover, and intelligent routing. They also publish the OpenRouter Agent SDK (`@openrouter/agent`) for building agents.
