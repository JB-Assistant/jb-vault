**Key insight:** Claude Design is a completely separate platform from all the other Claude tools (with its own usage limits).

## Section II: Accessing Claude Design in 60 Seconds

- Open Claude on the web, navigate to the left sidebar, find "Design"
- Or go directly to: https://claude.ai/design

**Usage Limits:**
- Claude Design runs on its own separate usage allowance
- Does not count against regular Claude chat or Claude Code limits
- Pro subscribers get a weekly allowance of Design prompts
- Max subscribers get a significantly higher limit with priority access
- Design credits reset weekly (unlike normal Claude chat which resets every few hours)
- **Use your Design credits carefully!**

## Section III: Using Claude Design

**The Interface:**
- **Left side** = Chat panel (describe what you want, iterate, give direction)
- **Right side** = Canvas (live, interactive workspace)

The homepage shows recent projects, designs, real examples, and design systems.

### Starting a Design

Four options in the left sidebar:
1. **Prototype** — Clickable, interactive design that simulates a real product
2. **Slide Deck** — Presentation built slide by slide
3. **From Template** — Pre-built starting point
4. **Other** — For everything else

Recommended: Select "Other," name the Project, select "Create." A new Project Design Canvas appears where you can start building in real-time.

You can start from scratch with natural prompts or add context (import data, Figma files, codebases, screenshots, and more). Remember: the more context, the better.

### Design Systems (Most Powerful Feature)

Design Systems are where you set guidelines and rules for specific visuals. Think of them like **Claude Skills, but for visuals** — a place where Claude knows your preset guidelines/instructions.

You can add brand colors, fonts, codebases, notes, and more. 

**Pro tip:** Create a Claude Skill first, then paste it into the "Other Notes" section. Example: copying .MD brand guidelines text into the Notes section means Claude Design now knows all the creative design components for your brand.

Once you create a Design System, select it for projects from the homepage. For example: "Slide Deck" + "AI Edge Design System."

Inside Design Systems you'll find:
- **Design System** — The instructions for the System
- **Design Files** — All the files within the System
- **Published/Default** — Where you can publish the System and set it as Default for all future Projects

Two methods for creating new designs:
- **New Project** — Start from scratch
- **Design Systems** — Pre-loaded instruction sets

## Section V: Pro Tips (Rapid-Fire)

- **Learn about Claude Design:** On the right side of the Canvas, select "Learn about Claude Design" for personalized tips
- **Approve/Deny:** Inside projects, tell Claude what "Looks Good" and what "Needs Work" — Claude learns what good visuals look like for you
- **Share:** In the top right of designs, export to Canva, add teammates, and more
- **Tweaks:** Easy way to edit visuals
- **Draw:** Sketch inside designs for hand-drawn visuals
