# Entangled Agents

Entangled agents are agents that co-evolve with the organizations they serve by observing work, canonicalizing stable facts into memory, creating skills from repeated approaches, and encoding repeated sequential patterns as deterministic flows.

## Loop

1. Usage produces conversational and operational traces.
2. Dreaming/review cycles canonicalize stable facts into memory.
3. Repeated approaches become skills.
4. Repeated sequences become deterministic flows.
5. Skills and flows improve future agent behavior.
6. Human review filters decide what gets accepted.

## Production Requirements

- Provenance trails for decisions and memories
- Memory aging/staleness awareness
- Lightweight execution paths
- Review filters for generated rules
- Trust-building behavior: admit gaps, avoid noisy narration, self-correct quietly
