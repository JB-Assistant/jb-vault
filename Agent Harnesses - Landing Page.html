What is an Agent Harness?

Everything that wraps the LLM and turns it from a token generator into a working agent. The model is commoditizing. The harness is the moat.

⚡

Tool Layer

Purpose-built tools for LLMs — not human APIs. 50+ custom tools spanning terminal, file, browser, database, and third-party APIs.

🧠

4-Layer Memory

Built-in memory → AGENTS.md/Soul.md → Obsidian Vault → Session Search. Progressive disclosure. No context drift.

🔒

Sandboxed Execution

Permission-gated terminal and file operations. Every command verified before execution. Audit trails built in.

🤖

Multi-Agent Coordination

Delegate tasks to parallel subagents. Each with isolated context, shared vault access, and coordinated handoffs.

📊

Evals & Tracing

Built-in evaluation suite. Every failure becomes a permanent fix — lint rule, skill, or context pattern.

🔀

Model Routing

Route tasks to the right model: GLM-5.1 for cost, DeepSeek V4 for reasoning, Claude for code. Optimized per-task economics.
